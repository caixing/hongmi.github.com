冒泡排序

11 January 2013

冒泡排序，通过重复交换相邻的两个反序元素来实现排序。 伪码如下（参考算法导论）

bubble-sort(A)
  for i =11 to len(A)
    for j = len(A) downto i + 1
      if A[j] < A[j - 1]
        then A[j] <-> A[j - 1]

内层循环每一次就将i..len(A)中的最小数交换到第i个位置，最终实现全部排序。
